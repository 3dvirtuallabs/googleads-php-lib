ConvertObjectToXml(mixed $object, string $rootElementName, bool $useXsiType) : string

Converts an object to an XML string.

The root element name is passed in as a parameter, and each field of the object becomes a child element. Array values are represented by multiples instances of that element. Methods on the object are ignored.
