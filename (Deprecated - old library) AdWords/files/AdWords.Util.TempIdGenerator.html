    A temporary ID generator produces decreasing negative integers.

    They can be assigned to AdWords entities (Operands) to create dependency relationships between them in Operations that are uploaded to the upload URL of BatchJobService.

    Generate() : integer
    static

    Generates a negative temporary ID which is a decreasing negative integer starting from -1.

    The function returns negative integers until reaching minimal integer value, which an exception is thrown instead.
